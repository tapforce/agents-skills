To install any component / present into project, MUST run command with `pnpm` and use this pattern:

```bash
pnpm dlx shadcn-svelte@latest add https://svelte-ai-elements.vercel.app/r/[COMPONENT].json
```

where `latest` should replace with the version of `shadcn-svelte` that is used in project.
