## Add shadcn components to project as needed

Always check component page from official site to see if any component match with requirements when write UXUI.

The official site located at: [Shadcn svelte components page](https://www.shadcn-svelte.com/docs/components).

## How to add component to project

- Use command `pnpm dlx shadcn-svelte@latest add <component-name>` to add component to project.
- Remember to replace `latest` with shadcn version installed in `package.json`. Lock version with project.

## Keep Shadcn's component files as it is, never change content.

Keep Shadcn's component files as it is, never change content. Using component in project following component document page from official site.
