**ALWAYS** respect Svelte 5 features when customizing styles, creating custom components, or using shadcn components.

**ALWAYS** respect TailwindCSS v4 features for all styling and customization work.

**NEVER** mimic old Svelte version styles or old Tailwind version patterns.

**ALWAYS** use modern syntax and patterns appropriate for Svelte 5 + TailwindCSS v4 combination.
