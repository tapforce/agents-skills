# Firebase Setup References

This directory contains reference documentation for Firebase integration with SvelteKit.

## Key Documentation

- [Firebase Web Setup Guide](https://firebase.google.com/docs/web/setup) - Main setup documentation
- [Firebase Config Object](https://firebase.google.com/docs/web/learn-more#config-object) - Configuration details
- [Firebase SDK Initialization](https://firebase.google.com/docs/web/setup#add-sdks-initialize) - App initialization

## Version Requirements

- Firebase SDK: >= 12.0.0
- SvelteKit: ^2
- Svelte: ^5
- TypeScript: Required
