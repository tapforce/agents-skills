# TailwindCSS v4 Framework Integration Rules

- **ALWAYS** follow framework-specific CSS file locations
- **ALWAYS** verify CSS imports are in correct framework entry points
- **NEVER** mix TailwindCSS v3 and v4 patterns in same project
